B.3 Compiling testfed
The tstfed executable file (called testfed.exe for Windows platforms and testfed for Linux platforms) is placed in the directory EXATA_HOME/bin when the Federation Interfaces Library is installed. This executable is recreated whenever EXata is recompiled with VR-Link.
If you need to recreate the testfed executable file, recompile QualNet with VR-Link (see Section 2.1.5 for details).

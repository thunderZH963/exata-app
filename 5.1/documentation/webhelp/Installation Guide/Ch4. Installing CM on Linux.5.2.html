4.1 Supported Platforms
EXata Connection Manager is supported on 32-bit (x86 compatible) and 64-bit (x86-64 compatible) platforms running one of the following Linux distributions:
•
CentOS 5.9
•
Red Hat Enterprise Linux 5.9
•
Ubuntu 12.04 LTS
 
Note: 
If you need to run EXata Connection Manager on a Linux platform not listed above, contact support@scalable-networks.com.
